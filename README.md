# Top Industrial

High-conversion bilingual (EN/ES) industrial sales website for Dongguan Top Industrial and Commerce Co. Ltd / 东莞若希迈贸易有限公司.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- next-intl (English / Spanish)
- Server Actions for Contact and RFQ forms

## Develop

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) (redirects to `/en`).

## Build

```bash
npm run build
npm start
```

## Positioning

Industrial Solutions Partner — engineering, execution, and a qualified manufacturing ecosystem. Not a trading company, sourcing agent, or broker.
